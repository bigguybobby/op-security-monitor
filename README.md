# OP Security Monitor

Real-time smart contract security monitoring and alerting dashboard for the Optimism ecosystem. Built for Optimism Season 9 Audit Grants.

## Features
- Real-time contract monitoring on OP Mainnet
- Automated vulnerability detection (reentrancy, access control, oracle manipulation)
- Alert system (Telegram, Discord, email)
- Audit report generator
- Historical security incident tracker

## Tech Stack
- **Frontend:** Next.js, TypeScript, Tailwind CSS, wagmi/viem
- **Backend:** Node.js, event listeners
- **Chain:** Optimism (OP Mainnet + Sepolia)
- **Analysis:** Slither integration, custom heuristics
